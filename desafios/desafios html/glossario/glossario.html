HTML

HTML (HyperText Markup Language) é uma linguagem descritiva que especifica a estrutura de uma página web.

Conceito e sintaxe

Um documento HTML é um documento de texto simples estruturado com elementos. Elementos são acompanhados de abertura e fechamento de tags. Cada tag começa e termina com colchetes angulares (< >). Existem algumas tags vazias ou sem conteúdo que não podem incluir qualquer texto, como por exemplo a tag .
